# OSINT Enigmas

Este repositório contém um site estático (HTML/CSS/JS) com vários enigmas para praticar técnicas de OSINT e decodificação (Morse, Binário, Base64, César).

## Estrutura
- `index.html` — página principal.
- `styles.css` — estilos.
- `script.js` — lógica dos enigmas e ferramentas.

## Personalização
- Adicione novos objetos no array `puzzles` dentro de `script.js`.
- Ajuste pontuação, dicas, e tipos de enigma conforme desejar.

## Avisos
-- Um dos enigmas (tipo `osint`) requer pesquisa externa e portanto não é validado localmente para o valor exato. Use fontes públicas e cite suas fontes ao usar em competições.
-- Eu não sei se o progresso tá salvando corretamente, ainda tô aprendendo sobre isso.
